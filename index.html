IBC Relayer

An IBC protocol-based relayer that allows 2 blockchains to transfer data and value between them. Think of this as TCP/IP (i.e. HTTP requests) for blockchains.

Horcrux

A threshold signer for Tendermint-based validators with a Harry Potter inspired name. This software separates validation keys into a number of pieces and allows the validator to make a signature from the sharded key for added security and greater reliability.

ibctest

ibctest orchestrates Go tests that utilize Docker containers for multiple IBC-compatible blockchains.
